The Office Filter

We will be using regex to improve our search results.

First, we'll check if any of the names includes the input. If it does, skip regex. Otherwise, move on to regex.

Our regex will check the first if any of the input letters match the first three letters of any name (in any order).

For example, if someone types 'pam', they will return 'Pam' in the search results without using our regex.

If, however, they type 'apm' it's very possible that they MEANT to type 'pam'.

Just because the literal string 'apm' is not in our collection doesn't mean we shouldn't render anything.
